## 生成 Word（.docx）

在**仓库根目录**执行：

```bash
pip install -r scripts/requirements-manual-docx.txt
python scripts/md_to_docx_manual.py
```

- 输出目录：`docs/standard-manual/word/`
- 单文件生成：`python scripts/md_to_docx_manual.py docs/standard-manual/01-求职者操作手册.md`
- 图片相对路径：手册中引用 `../video_frames/frame_*.png`，请保证该目录存在（可由录屏按 `docs/录屏与截图分析.md` 中命令重新抽帧）。

**系统管理员**分册当前无录屏配套截图，Word 中仅文字；如需插图可在后台截取「角色管理 / 用户管理 / 部门管理」后粘贴到 docx。
